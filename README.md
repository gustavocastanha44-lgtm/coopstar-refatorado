# Coopstar Express - Refatoração Moderna 🚀

Este projeto é uma refatoração completa e modernização do site da **Coopstar Express**, transformando uma estrutura legada em uma aplicação web de alta performance utilizando as tecnologias mais recentes do mercado.

## 🛠️ Tecnologias Utilizadas

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações:** [Framer Motion](https://www.framer.com/motion/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Imagens:** Otimizadas e geradas por IA para uma estética premium.

## ✨ Principais Funcionalidades

- **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktop.
- **Performance Extrema:** Utiliza renderização estática e otimização de imagens do Next.js.
- **Identidade Visual Preservada:** Foco na cor vermelha da marca (`#CC0000`).
- **Animações Fluidas:** Transições suaves de scroll e entradas de elementos.
- **Mapa Integrado:** Google Maps funcional apontando para a sede em Moema.
- **Formulário de Contato:** Integração direta com o WhatsApp para conversão imediata.

## 🚀 Como Executar o Projeto

### Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/gustavocastanha44-lgtm/coopstar-refatorado.git
   ```

2. Entre no diretório:
   ```bash
   cd coopstar-refatorado
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

### Desenvolvimento

Para rodar o servidor de desenvolvimento local:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## 📦 Build para Produção

Para gerar uma versão otimizada para produção:

```bash
npm run build
```

E para iniciar o servidor de produção:

```bash
npm run start
```
